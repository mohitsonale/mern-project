# MERN Authentication App
A full-stack MERN (MongoDB, Express, React, Node.js) authentication project with JWT based login, registration, and protected routes. This project demonstrates secure user authentication, cookie handling, and responsive UI with React + Tailwind CSS.

# Features

-User registration and login

-JWT authentication with cookies

-Protected routes (accessible only to logged-in users)

-Responsive UI built with React & Tailwind CSS

-MongoDB Atlas for database storage

-Environment variables for secure configuration

-Ready for production deployment

# Tech Stack
-Frontend: React, Tailwind CSS, Axios

-Backend: Node.js, Express.js

-Database: MongoDB (Atlas)

-Authentication: JWT, Cookies

-Other Tools: dotenv, cookie-parser, CORS
